Google Drive — nastavenie
Odporúčané riešenie pre cloudovú synchronizáciu
🔧 Prečo Google Drive?
• Bezplatné úložisko 15 GB · prístup z PC, mobilu, tabletu
• Zabudované OCR pre naskenované (papierové) správy
• Google Apps Script = bezplatný backend bez servera
• Claude API integrácia pre automatickú extrakciu obsahu
Kroky nastavenia
1
Vytvorte priečinok v Google Drive
Napr. MedArchív/Správy — sem nahráte PDF súbory
2
Nainštalujte Google Drive pre PC
Synchronizuje lokálne PDF priamo do cloudu. Drag&drop z Prieskumníka súborov.
3
Naskenujte papierové správy
Použite aplikáciu Google Drive na mobile (zabudovaný skener + OCR v slovenčine)
4
Claude API — automatická extrakcia
Apps Script spustí Claude Sonnet, extrahuje dátum/odbor/lekára/dg./medikáciu a uloží do Google Sheets (databázy katalógu)
5
Publikujte katalóg ako Web App
Apps Script → Nasadiť ako webová aplikácia → prístup cez URL z ľubovoľného zariadenia
💡 Tip: Požiadajte Claude (v tejto konverzácii) o vygenerovanie kompletného Google Apps Script kódu pre váš katalóg vrátane Claude API integrácie. Stačí napísať: "Vygeneruj Apps Script kód pre MedArchív s mojím Google Drive"
